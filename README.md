Project Description (Suggested)

Kenya Counties, Sub-Counties and Wards is a comprehensive, structured, and developer-friendly dataset containing all administrative divisions of Kenya.
It includes:

47 Counties

290+ Sub-Counties

1,450+ County Assembly Wards

The dataset is designed to support developers, researchers, data analysts, GIS projects, and organizations building location-aware applications in Kenya.
It provides clean, normalized, and ready-to-use data formats.

Features

Complete list of counties, sub-counties, and wards

Hierarchical structure (County → Sub-County → Ward)

Machine-readable formats (JSON, CSV, SQL, YAML, etc.)

Unique standardized codes

Consistent naming conventions

Easy integration into web, mobile, and backend systems

Regular updates as administrative boundaries evolve

Ideal Use Cases

Government service apps

Logistics & delivery systems

Fintech (location KYC, addressing)

GIS & mapping platforms

Research & demographic analytics

E-commerce location mapping

Polling & electoral boundary tools
